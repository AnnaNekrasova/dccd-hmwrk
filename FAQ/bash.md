# Основные команды bash (терминала)

```sh
$ pwd # Выводит в терминал текущее местоположение в файловой системе
$
$ ls -l # Выводит в терминал все файлы и папки в текущем местоположении
$
$ cd Desktop # Переходит в папку Desktop (папку можно заменить на любую существующую в текущем местоположении)
$
$ mkdir new-project # Создаёт папку в текущем местоположении в файловой системе
```
